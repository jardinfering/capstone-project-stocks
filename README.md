# Capstone Project Stock Data Analysis

Stock Data Analysis Capstone Project
Overview

This project is an interactive stock data analysis tool that allows users to retrieve financial information, generate graphs, and perform basic data analysis on selected stocks. It uses real market data to provide insights such as historical prices, financial statistics, and comparisons between multiple stocks.

Features
Display historical stock price graphs for a selected time period
Retrieve full financial statistics for a stock
Query specific financial metrics such as market cap, P/E ratio, and dividend yield
Compare price trends of multiple stocks on the same graph
Interactive menu-driven user interface
How It Works

Users interact with a menu system to select different options. Depending on the selection, the program requests stock tickers, date ranges, or specific metrics, then processes the input and returns either visual graphs or financial data.

Input Requirements
Stock ticker symbols (e.g., AAPL, TSLA)
Date ranges for analysis
Graph function: DD-MM-YYYY
Comparison function: YYYY-MM-DD
Financial metric names for single-metric queries
Menu selections (1–5) to choose program functions
Yes/No input to continue or exit
Technologies Used
Python
yfinance (financial data API)
pandas (data manipulation)
numpy (numerical processing)
matplotlib (graphing)
seaborn (visual styling)
Project Purpose

This project was built as a capstone to demonstrate skills in data handling, user input processing, visualization, and applying programming concepts to real-world financial data.
